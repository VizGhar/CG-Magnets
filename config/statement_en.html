The Goal

You are given a board where you must place magnets so that all rules are satisfied and the final configuration is stable.

  Rules

Each magnet consists of two parts: a blue north pole (-) and a red south pole (+). You have an unlimited supply of such magnets. Your task is to place the required magnets on the board so that the following conditions are met:

  • Magnets may only be placed on predefined cells.
  • Identical poles (cells with +, -) can't touch each other making the board unstable.
  • The numbers on the left side of the board (leftHint) indicate the exact number of + poles that must appear in each row.
  • The numbers above the board (topHint) indicate the exact number of + poles that must appear in each column.
  • The numbers on the right side of the board (rightHint) indicate the exact number of - poles that must appear in each row.
  • The numbers below the board (bottomHint) indicate the exact number of - poles that must appear in each column.
  • Not all cells need to contain magnets; some might be occupied by wooden blocks marked with x.

  Game Input

Initial input

Line 1: single integer width, the width of the board.

Line 2: single integer height, the height of the board.

Line 3: height space-separated integers leftHint (number of + poles in row i).

Line 4: width space-separated integers topHint (number of + poles in column i).

Line 5: height space-separated integers rightHint (number of - poles in row i).

Line 6: width space-separated integers bottomHint (number of - poles in column i).

Note: You will receive -1 if there is no hint specified for given row or column.

Turn input

Next height lines: each line contains a string of width characters:

  • + → cell occupied by the south pole of a magnet.
  • - → cell occupied by the north pole of a magnet.
  • x → cell occupied by wooden block.
  • Letter (A–Z, a–z) marking predefined magnet cell pairs. Identical letters mark two adjacent cells that belong to the same magnet.

Turn output

Single line: three space-separated values - integers x and y and character symbol (one of +, -, x).
This output will place symbol at [x, y], the coordinates of a cell (the matching cell is placed automatically).

Constraints
6 ≤ width ≤ 10
5 ≤ height ≤ 9

Allotted response time to output in first turn is ≤ 5s
Allotted response time to output in other turns is ≤ 50ms
